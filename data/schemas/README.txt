SCHEMA FOLDER - Database Structures
====================================

Files:
- master_schema.sql     : Master bot tables (system_bots, permissions, logs)
- ardayda_schema.sql    : Ardayda bot tables (users, pdfs, tags, admin_logs)
- dhalinyaro_schema.sql : Dhalinyaro bot tables (coming soon)

Usage:
- These are backup copies of your table structures
